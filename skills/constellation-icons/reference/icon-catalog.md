# Constellation Icon Catalog

**Package:** `@zillow/constellation-icons` v10.11.0
**Total Icons:** 621 (302 Filled + 302 Outline + 17 Special)

## Quick Import
```tsx
import { IconHeartFilled } from '@zillow/constellation-icons';
```

## Usage with Icon Component
```tsx
import { Icon } from '@zillow/constellation';
import { IconHeartFilled } from '@zillow/constellation-icons';

<Icon size="md"><IconHeartFilled /></Icon>

// Size tokens: sm (16px), md (24px), lg (32px), xl (44px)
// For semantic colors, use css prop (NOT color prop):
<Icon size="md" css={{ color: 'icon.neutral' }}><IconHeartFilled /></Icon>
```

## Rules
- **ALWAYS** use Filled variants by default
- **ALWAYS** wrap in `<Icon size="...">` with size tokens (sm/md/lg/xl)
- **NEVER** use Outline as default; reserve for secondary/inactive states
