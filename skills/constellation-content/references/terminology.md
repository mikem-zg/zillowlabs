# Zillow Terminology & Definitions

Source: [Zillow Style Guide — Terms](https://zillow.styleguide.com/terms-snippets-definitions/terms)

This glossary contains Zillow-specific terms with their approved usage, status, and notes. Always check this list before writing product copy.

## Status Key

| Status | Meaning |
|--------|---------|
| **Approved** | Use this term as documented |
| **Don't use** | Never use this term — see the approved alternative |
| **Use carefully** | OK in specific contexts — read the guidance |

---

## Don't Use (Banned Terms)

These terms must never appear in Zillow product copy. Use the approved alternative.

| Don't Use | Use Instead | Reason |
|-----------|-------------|--------|
| blacklist | block, blocked users | Rooted in racism and oppression |
| whitelist | approved | Rooted in racism and oppression |
| white glove | (describe the service directly) | Rooted in racism and oppression |
| master / master bedroom | primary / primary bedroom | Inclusive language |
| his-and-her | (describe without gendered framing) | Heteronormative |
| Jack-and-Jill | (describe without gendered framing) | Heteronormative, culturally specific |
| mother-in-law suite | (describe the space directly) | Ageist and sexist |
| walk-in closet | (use dimensions, e.g., 8' x 10' closet) | Ableist |
| walking distance | (use actual distance or travel time) | Ableist |
| walkthrough | (use "tour" or "video tour") | Ableist |
| click / click here | select (and name the target) | Device-agnostic language |
| tap | select (and name the target) | Device-agnostic language |
| disable | turn off, deactivate | Implies "disabled" is negative |
| enable | turn on, allow, show | Avoid when possible |
| stories (floors) | levels (homes), floors (apartments) | Ambiguous |
| login (verb) | sign in (verb), sign-in (noun/adj) | AP style |
| list price | listing price | "Listing price" is the standard term |
| instant offer | (do not use — product renamed) | No longer accurate |
| HDP | (do not use in consumer copy) | Internal abbreviation |
| iBuyer | (do not use in product copy) | Internal term |
| ZIBS / ZPAR | (use full names) | Internal abbreviations |
| okay | OK | Zillow standard |
| FAQs | FAQ | Never pluralize |

## Use Carefully

These terms are OK in specific contexts but require judgment.

| Term | Guidance |
|------|----------|
| affordability | Don't say "your affordability." Use "what you can afford" or "Is this home affordable for you?" |
| borrowing | We often use "finance" instead in headlines. Use judgment based on clarity and tone |
| client | Only when referencing buyers/sellers who have a clear relationship with an agent |
| got it | May confuse non-native speakers. Prefer "OK" for confirmation buttons |
| info | Try the full word "information." "Info" OK when space is limited. Consider "details" |
| lead | OK for Flex agents. Don't use when talking about home shoppers connected to Premier Agent partners who pay for advertising (RESPA issue) |
| low income | Only when followed by "family" or "household." Prefer "income restricted" |
| never / always | Legal advises against terms with no wiggle room. Avoid or check with Legal |
| please | Use sparingly — research shows it can come across as condescending |
| see / view | Some consider these ableist. Use more expressive verbs when possible (explore, display, check). Generally OK in digital contexts |
| sorry | Only when error is Zillow's fault AND causes significant inconvenience |
| tenant | Reserve for longtail Rentals copy. When in doubt, use "renter" |
| thank you / thanks | Save for when the user performs an action in response to a serious issue that was our fault |
